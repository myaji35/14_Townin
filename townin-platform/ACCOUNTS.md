## 👤 계정 목록

### 1. 슈퍼관리자 (Super Admin)

| 항목 | 내용 |
|------|------|
| **역할** | super_admin |
| **이메일** | admin@townin.kr |
| **비밀번호** | townin2025! |
| **권한** | 전체 시스템 관리, 모든 지역 접근, 자치체/보안관 임명 |
| **접근 가능 화면** | 모든 화면 + 시스템 설정 |

**주요 기능:**
- ✅ 전국 모든 지역 데이터 조회 및 관리
- ✅ 자치체 관리자 및 지역관리자(보안관) 임명/해임
- ✅ 시스템 통계 및 분석 (전국 단위)
- ✅ 마스터 데이터 관리 (grid_cells, safety_data 등)
- ✅ 전체 사용자/상인/전단지 관리
- ✅ 감사 로그 조회

### 2. 자치체관리 (Municipality Admin)

| 항목 | 내용 |
|------|------|
| **역할** | municipality |
| **이메일** | municipality@uijeongbu.go.kr |
| **비밀번호** | townin2025! |
| **담당 지역** | 경기도 의정부시 |
| **권한** | 의정부시 전체 데이터 관리, 지역 보안관 관리 |
| **접근 가능 화면** | 의정부시 대시보드, 지역 통계, 보안관 관리 |

**주요 기능:**
- ✅ 의정부시 전체 통계 대시보드
  - 총 사용자 수

  - 총 전단지 수

- ✅ 의정부시 소속 지역관리자(보안관) 관리
  - 보안관 임명/해임
  - 보안관별 성과 조회
- ✅ 의정부시 공공 안전 데이터 관리
  - CCTV, 가로등, 주차장 데이터 추가/수정
- ✅ 의정부시 상인 및 전단지 승인/거부
- ✅ 의정부시 지역 분석 리포트

## 🚀 계정 테스트 시나리오

### 시나리오 1: 슈퍼관리자
1. `admin@townin.kr`로 로그인
2. 전국 통계 대시보드 확인
3. 의정부시 자치체관리 계정 확인
4. 새로운 보안관 임명 테스트
5. 감사 로그 조회

### 시나리오 2: 자치체관리 (의정부시)
1. `municipality@uijeongbu.go.kr`로 로그인
2. 의정부시 전체 통계 확인
3. 지역별(동별) 현황 조회
4. 보안관 성과 리포트 확인
5. 새로운 안전 데이터 추가 (예: CCTV)

### 시나리오 3: 지역관리자 (의정부동 보안관)
1. `guard1@townin.kr`로 로그인
2. 이번 달 수익 현황 확인
3. 담당 구역 상인 목록 조회
4. 새로운 상인 등록 (테스트)
5. 전단지 홍보 성과 확인

### 시나리오 4: 일반 사용자
1. `user1@example.com`로 로그인
2. 거주 지역(의정부동) 안전 지도 조회
3. 오늘의 전단지 피드 확인
4. 전단지 조회 (₩25 포인트 적립)
5. 가족 케어 상태 확인

## 🔧 계정 비밀번호 변경

개발/테스트 후 프로덕션 배포 시 **반드시** 모든 계정의 비밀번호를 변경하세요.

```sql
-- 비밀번호 해시 생성 예시 (bcrypt)
-- 새 비밀번호: "your_new_password"
UPDATE users SET password_hash = '$2b$10$NEW_HASH_HERE' WHERE email = 'admin@townin.kr';
```
